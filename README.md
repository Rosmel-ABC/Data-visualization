
<h1 align="center">Visualización de Datos con Matplotlib y Seaborn</h1>

<p align="center">Este repositorio contiene una serie de ejercicios de visualización de datos utilizando las bibliotecas de Python, Matplotlib y Seaborn. Estos ejercicios están diseñados para ayudarte a mejorar tus habilidades en la representación gráfica de datos y explorar diferentes técnicas de visualización.</p>

---

## Descripción

La visualización de datos es una parte fundamental en el análisis de datos y la comunicación efectiva de resultados. Matplotlib y Seaborn son dos bibliotecas populares de Python que brindan una amplia gama de herramientas y opciones para crear gráficos y visualizaciones de datos.

En este repositorio, encontrarás una serie de ejercicios prácticos que te permitirán practicar y familiarizarte con las funcionalidades de Matplotlib y Seaborn. Cada ejercicio incluye un conjunto de datos de ejemplo y una descripción de la tarea a realizar.

## Contenido
Cada  archivo Jupyter Notebook continene varios tipos de gráficos. Puedes abrir los archivos `.ipynb` directamente en tu entorno de desarrollo de Jupyter Notebook o ejecutarlos en un servidor de notebook en la nube.

Las gráficas elaboradas son las siguientes:

1. **Gráfico de funciones trigonométricas**
2. **Gráfico de dispersión**
3. **Gráficos de barras**
4. **Histogramas**
5. **Gráfico Forecasting**
6. **Gráficos con Seaborn**


## Librerías Utilizadas

- Matplotlib
- Seaborn
- NumPy
- Pandas
